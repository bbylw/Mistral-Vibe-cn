CLI 介绍

Mistral Vibe 是一个由 Mistral AI 模型驱动的命令行编程助手。它为您的代码库提供了一个对话式界面，让您可以使用自然语言来探索、修改和与您的项目进行交互。

源码托管在 GitHub 上，采用 Apache 2.0 开源许可证。

核心特性

💬

交互式对话

一个智能的 AI 代理，能够理解您的请求并分解复杂的开发任务。

🛠️

强大工具集

内置文件操作、代码搜索、版本控制和命令执行等一系列强大工具。

📂

项目上下文感知

自动扫描文件结构和 Git 状态，为 AI 提供最相关的代码上下文。

⚡

高级 CLI 体验

支持斜杠命令、文件路径自动补全以及持久化命令历史。

内置工具能力

  • 文件操作：读取、写入和修补文件 (read_file, write_file, search_replace)。
  • 命令执行：在有状态的终端中执行 shell 命令 (bash)。
  • 代码搜索：使用 grep 递归搜索代码（支持 ripgrep）。
  • 待办管理：管理待办事项列表以跟踪代理的工作进度。

安全优先

Mistral Vibe 内置了严格的工具执行审批机制。在执行任何可能修改代码或系统的操作前，都会先行展示预览并征得您的确认。
