安装指南

Mistral Vibe 是一个 Python 包，您可以直接在终端中安装并作为命令行工具使用。

适用人群

  • 终端优先的开发者。
  • 希望 AI 助手能与其本地环境交互的用户。
  • 需要可复现、脚本友好型命令行工作流的团队。

快速安装

在您的终端中运行以下命令即可快速安装：

curl -LsSf https://mistral.ai/vibe/install.sh | bash

其他安装方式

使用 uv (推荐)

如果您已经安装了 uv，可以使用以下命令：

uv tool install mistral-vibe

如果您在使用 uv 时遇到问题，尝试添加 --no-cache 标志：
uv tool install mistral-vibe --no-cache

使用 pip

请确保您的 Python 版本为 3.12+：

pip install mistral-vibe

平台支持

Mistral Vibe 可以在 Windows 上运行，但官方主要支持和针对 UNIX 环境进行优化。

源码与贡献

您可以克隆 GitHub 仓库进行本地安装或自定义开发：

git clone https://github.com/mistralai/mistral-vibe.git
cd mistral-vibe
